# RacketStream Video Library web app

## Live link: [RacketStream](https://racketstream.netlify.app/)

## Features:

1. Auth (Login & Signup)
2. Private Routes
3. Video Page
4. Liked Videos
5. Watch Later Videos
6. Playlist Management (Create or Delete User Playlists)
7. Add/Remove videos to/from playlist
